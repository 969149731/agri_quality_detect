特色果蔬重金属超标情况建议


特色蔬菜主要的风险因子集中在重金属方面。为了减少叶菜类蔬菜的重金属污染，建议如下：👇
  1. 采取网格化管理，在土地本底重金属超标的区域严禁种植果蔬，加强对蔬菜种植区的重金属监测力度。
  2. 避免在污染土壤种植易富集重金属的蔬菜品种/类型，同时采取一定的措施减低重金属活性。
  3. 改变蔬菜的种植技术，合理使用化肥农药，上市之前做好蔬菜的重金属含量检测。


推动全产业链监管体系升级


特色小宗果蔬产业规模有限，大型基地较少，多数是以小农户家庭生产方式为主。建议如下：👇
  1. 建立一批广西特色蔬菜的标准化生产基地，培育特色蔬菜品牌，加大对全产业链的监管力度，建立健全全产业链监管体系升级。
  2. 建议建立各县风险评估站，实时实地监控关键风险因子。


⭐特色果蔬质量安全评估建议



推动建立风险评估，全力保障消费者权益


广西特色果蔬农药监测总体的合格率较高，风险较小，但是超范围使用农药的现象仍存在。现有的国家标准对于南瓜花、南瓜苗、豌豆苗等南方常见蔬菜无明确的农药使用限量。农业部门主要的监管集中在大宗蔬菜，对特色小宗果蔬质量安全的监管不足，属于监管的空白。建议政府加强对小宗特色果蔬风险评估及监管，制定有效安全用药规范，推动建立风险评估体系。

部分特色野菜在套种模式安全性建议


部分特色蔬菜常与果树套种，这种立体种植模式可提高土地的复种指数和资源利用率，但使用农药时需要考虑到果树规定可使用的农药在蔬菜种植中可能是限用或禁用药，因此建议如下：👇
  1. 选择病虫害较少的两种作物，避免共同病虫害作物套种，减少农药的使用。
  2. 套种的时间错开，蔬菜的喷药期应当考虑果树的安全间隔期。
  3. 尽量分开种植。


推进风险评估技术发展


建议如下：👇
  1. 建立更加完善的科学创新技术评估体系，加强风险评估技术创新，有利于提升特色小宗果蔬的质量和安全水平，促进特色小宗果蔬产业的发展。
  2. 可通过收集和整理特色小宗果蔬风险评估相关数据，构建特色小宗果蔬风险评估风险等级数据库，以此实现在第一时间评估对应风险，准确发布权威数据，应对舆情风险。
